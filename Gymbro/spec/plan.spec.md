# Epic
Create a plan with exercises to be performed in some day of the week. 
What is not relevant now:
 - In which day of the week the exercise will be performed.

## Requisites
- The user must be able to create a list with an arbitrary number of exercises from the database provided to the application.
- The exercises consist in a card with the following information: name, force, level, mechanic and equipment
- If the user taps the exercise card a modal is opened where, if there exists images for the exercise they are shown in sequence giving the idea of a gif and the instructions are detailed.
- Note: the performance of the user is not to be created here this is because the performance in an exercise might vary according to the position the exercise takes in the plan
